Privacy Policy

ChatRipper AI Chrome Extension — Effective March 9, 2026

1. Introduction

ChatRipper AI is a Chrome extension developed by Martell Media for internal use by its revenue team. It provides AI-powered reply suggestions across supported messaging platforms.

2. Data Collection

The extension accesses the following data when you use it:

  • Conversation text from supported messaging platforms
  • Contact details (name, ID, channel type)
  • CRM metadata (sales pipeline data, notes, tags)
  • User settings (engine preference, API key — stored locally in your browser)

The extension does not collect:

  • Passwords or login credentials
  • Financial information
  • Data from websites outside the supported platform list

3. How Data Is Used

  • Conversation text is sent to backend AI services to generate reply suggestions.
  • Contact metadata is used to personalize suggestions.
  • Your API key authenticates requests and is never sent to third parties.

4. Data Storage

5. Third-Party Services

  • Some reply suggestions are processed by third-party AI services operated independently from Martell Media.
  • These services receive conversation data for processing.
  • Data retention practices for these services have not been independently verified.
  • No data is sold or shared with advertisers.

6. Data Security

  • All data is transmitted over HTTPS.
  • API keys are stored in Chrome's browser-managed local storage, protected by your OS login and browser profile.
  • Per-user authentication — each user has a unique key.
  • The extension is restricted to specific website domains.

7. User Rights

This is an internal tool. For data requests, contact Alfred Loh at alfred@danmartell.com.

  • You can clear your API key and settings via the extension.
  • Uninstalling the extension removes all locally stored data.
